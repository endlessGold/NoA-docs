# 1. 기능 개요
ConvertTo-WslPath는 Windows 경로를 WSL 경로로 변환합니다.

# 2. 도메인 맥락
WSL에서 emcc를 호출할 때 Windows 경로를 리눅스 경로로 변환하기 위해 사용됩니다.

# 3. 기능 요구사항
1) 드라이브 문자를 /mnt/<drive> 형식으로 변환해야 한다.
2) 역슬래시를 슬래시로 변환해야 한다.

# 4. 입력 정의
- Path: string

# 5. 출력 정의
- return: string

# 6. 비기능 요구사항
1) 입력이 이미 리눅스 경로인 경우 그대로 반환해야 한다.

# 7. 경계 조건 및 제약
1) 입력이 빈 문자열일 때의 처리 규칙이 정의되어야 한다.

# 8. 미정 사항
1) UNC 경로 지원 여부

[논리적 검증 결과]
- 누락 항목: UNC 경로 처리, 빈 문자열 정책
- 충돌 항목: 없음
- 모호한 항목: 리눅스 경로 판별 기준

## 예시

```javascript
const wsl_path = ConvertTo_WslPath("./NoA");
```
