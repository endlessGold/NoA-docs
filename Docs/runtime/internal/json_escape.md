# 1. 기능 개요
json_escape는 입력 문자열을 JSON 안전 문자열로 이스케이프합니다.

# 2. 도메인 맥락
NoA 런타임이 JSON 응답 문자열을 안전하게 구성하기 위해 사용됩니다.

# 3. 기능 요구사항
1) 따옴표, 역슬래시, 제어 문자를 이스케이프해야 한다.
2) null 입력을 허용해야 한다.

# 4. 입력 정의
- value: const char*

# 5. 출력 정의
- return: char* (이스케이프된 문자열 또는 null)

# 6. 비기능 요구사항
1) 메모리 할당 실패 시 null을 반환해야 한다.

# 7. 경계 조건 및 제약
1) value가 null이면 빈 문자열을 반환해야 한다.

# 8. 미정 사항
1) 유니코드 이스케이프 처리 범위

[논리적 검증 결과]
- 누락 항목: 유니코드 처리 범위
- 충돌 항목: 없음
- 모호한 항목: 제어 문자 확장 범위

## 예시

```javascript
const escaped = json_escape("text\"with\"quotes");
```
