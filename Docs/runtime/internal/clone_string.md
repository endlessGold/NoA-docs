# 1. 기능 개요
clone_string은 입력 문자열을 복제해 새 메모리를 반환합니다.

# 2. 도메인 맥락
NoA 런타임 내부에서 안전한 문자열 소유권을 확보하기 위해 사용됩니다.

# 3. 기능 요구사항
1) null 입력을 허용해야 한다.
2) 입력 문자열 길이에 맞는 새 버퍼를 할당해야 한다.

# 4. 입력 정의
- value: const char*

# 5. 출력 정의
- return: char* (복제된 문자열 또는 null)

# 6. 비기능 요구사항
1) 메모리 할당 실패 시 null을 반환해야 한다.

# 7. 경계 조건 및 제약
1) value가 null이면 null을 반환한다.

# 8. 미정 사항
1) 최대 문자열 길이 제한

[논리적 검증 결과]
- 누락 항목: 길이 제한 정책
- 충돌 항목: 없음
- 모호한 항목: 메모리 실패 시 상위 처리 정책

## 예시

```javascript
const copy = clone_string("text");
```
