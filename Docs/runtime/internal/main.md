# 1. 기능 개요
main은 표준 입력을 수신해 NoA JSON 실행 경로를 호출합니다.

# 2. 도메인 맥락
CLI 또는 파이프 입력을 통해 NoA 런타임을 구동합니다.

# 3. 기능 요구사항
1) 표준 입력을 읽어 JSON 요청을 생성해야 한다.
2) JSON 처리 함수를 호출해 응답을 출력해야 한다.

# 4. 입력 정의
- 표준 입력 JSON 문자열

# 5. 출력 정의
- 표준 출력 JSON 문자열

# 6. 비기능 요구사항
1) 오류 발생 시에도 JSON 형식으로 응답해야 한다.

# 7. 경계 조건 및 제약
1) 입력 읽기 실패 시 오류 응답을 출력해야 한다.

# 8. 미정 사항
1) 종료 코드 규칙

[논리적 검증 결과]
- 누락 항목: 종료 코드 정책
- 충돌 항목: 없음
- 모호한 항목: 오류 발생 시 출력 형식 고정 여부

## 예시

```javascript
const input = read_all_stdin();
const output = noa_run_json(input);
console.log(output);
```
