# 1. 기능 개요
read_all_stdin은 표준 입력을 모두 읽어 문자열로 반환합니다.

# 2. 도메인 맥락
CLI 모드에서 JSON 입력을 수신하기 위해 사용됩니다.

# 3. 기능 요구사항
1) 입력 전체를 읽어 null 종료 문자열로 반환해야 한다.
2) 입력 길이에 따라 버퍼를 확장해야 한다.

# 4. 입력 정의
- 표준 입력 스트림

# 5. 출력 정의
- return: char* (입력 문자열 또는 null)

# 6. 비기능 요구사항
1) 메모리 할당 실패 시 null을 반환해야 한다.

# 7. 경계 조건 및 제약
1) 빈 입력도 정상적으로 처리해야 한다.

# 8. 미정 사항
1) 최대 입력 크기 제한

[논리적 검증 결과]
- 누락 항목: 최대 입력 크기 제한
- 충돌 항목: 없음
- 모호한 항목: 빈 입력 처리 규칙

## 예시

```javascript
const input = read_all_stdin();
```
