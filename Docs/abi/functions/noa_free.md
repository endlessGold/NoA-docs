# 1. 기능 개요
noa_free는 NoA ABI에서 반환한 메모리를 해제하는 함수입니다.

# 2. 도메인 맥락
noa_run_json 등에서 반환된 동적 메모리 해제를 위한 표준 API입니다.

# 3. 기능 요구사항
1) null 포인터 입력을 허용해야 한다.
2) 반환 메모리를 정상적으로 해제해야 한다.

# 4. 입력 정의
- ptr: char*

# 5. 출력 정의
- 반환값 없음

# 6. 비기능 요구사항
1) 호출 시 예외나 크래시가 발생하지 않아야 한다.

# 7. 경계 조건 및 제약
1) ptr이 null이면 아무 동작도 하지 않아야 한다.
2) ptr은 NoA ABI가 반환한 메모리여야 한다.

# 8. 미정 사항
1) 이중 해제에 대한 보호 정책

[논리적 검증 결과]
- 누락 항목: 이중 해제 방지 정책
- 충돌 항목: 없음
- 모호한 항목: 타 allocator 사용 시 처리

## 예시

```javascript
const output_json = noa_run_json(input_json);
noa_free(output_json);
```
